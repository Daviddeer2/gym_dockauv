\[\begin{split}\boldsymbol{J}_{\Theta}(\boldsymbol{\eta}) = \begin{bmatrix} \boldsymbol{R}_b^n(\boldsymbol{\Theta}_{nb}) & 0\\ 0 & \boldsymbol{T}_{\Theta}(\boldsymbol{\Theta}_{nb}) \end{bmatrix}\end{split}\]

gym_dockauv.utils.geomutils.Rzyx(phi, theta, psi)

Rotation matrix from {b} to {n}, to obtain a vector in NED coordinates that is expressed in the body frame, a matrix multiplication with the rotation matrix is used. For the reserve way, use the transposed rotation matrix. In the formula below abbreviations \(s\) for sin and \(c\) for cos are used.

\[\begin{split}\boldsymbol{R}_b^n(\boldsymbol{\Theta}_{nb}) = \begin{bmatrix} c \psi c \theta & -s \psi c \phi + c \psi s \theta s \phi & s \psi s \phi + c \psi c \phi s \theta \\ s \psi c \theta & c \psi c \phi + s \phi s \theta s \psi & -c \psi s \phi + s \theta s \psi c \phi \\ -s \theta & c \theta s \phi & c \theta c \phi \end{bmatrix}\end{split}\]
Parameters

  • phi (float) – euler angle around the x axis

  • theta (float) – euler angle around the y axis

  • psi (float) – euler angle around the z axis

Return type

ndarray

Returns

array 3x3

gym_dockauv.utils.geomutils.Tzyx(phi, theta)

Angular transformation matrix from {b} to {n}. In the formula below abbreviations \(s\) for sin, \(c\) for cos and \(t\) for tan are used.

Note

Note that this transformation is not well-defined for \(\theta = \frac{\pi}{2}\), however we do not expect an AUV to do that. An alternative approach could be quaternions, but are not implemented here.

\[\begin{split}\boldsymbol{T}_{\Theta}^n(\boldsymbol{\Theta}_{nb}) = \begin{bmatrix} 1 & s \phi t \theta & c \phi t \theta \\ 0 & c \phi & -s \phi \\ 0 & \frac{s \phi}{c \theta} & \frac{c \phi}{c \theta} \end{bmatrix}\end{split}\]
Parameters

  • phi (float) – euler angle around the x axis

  • theta (float) – euler angle around the y axis

Return type

ndarray

Returns

array 3x3

Bases: object

Provide Low pass Filter calculations

property alpha: float

Smoothing factor for discrete time implementation, \(0 <= \alpha <= 1\). When \(\alpha=0.5\), the low pass time constant is equal to the sample time, if it is smaller than 0.5, the time constant is larger than then the sample time.

\[\alpha = \frac{\Delta_T}{T_1 + \Delta_T}\]
Returns

smoothing factor

apply_lowpass(x, y_prev)

Applies low-pass according to a discrete-time implementation of a simple low-pass filter:

\[y_i = \alpha x_i + (1 - \alpha) * y_{i-1}\]
Parameters

  • x (ndarray) – e.g. command at time i

  • y_prev (ndarray) – previous command at time i-1

Return type

ndarray

Returns
